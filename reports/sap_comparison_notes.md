# SAP Comparison Notes

## 1. Purpose

SAP S/4HANA Cloud Public Edition is used as a conceptual comparison layer.

This document does not claim SAP implementation.

## 2. Comparison Logic

ECOUNT practice is mapped to generic ERP processes and SAP-style concepts.

Examples:

| ECOUNT | Generic ERP | SAP-style |
|---|---|---|
| 발주서입력 | Purchase Order | Purchase Order |
| 구매입력 | Goods Receipt | Procurement Execution |
| 주문서입력 | Sales Order | Sales Order |
| 작업지시서입력 | Work Order | Production Order |
| BOM | Bill of Materials | BOM |

## 3. Limitation

Mappings are process-level and conceptual. They are not one-to-one app or configuration mappings.
