# erp-workflow-cwi-lab
This project uses one of the ERP demos which includes some workflows and synthetic event logs to study ERP workflow visibility, production/logistics processes, and CWI-style operational intelligence. SAP S/4HANA Cloud Public Edition is used as a conceptual comparison layer where direct practice access is limited.
Combinatorial Workflow Intelligence uses ERP event logs, binary factor encoding, and combinatorial scenario design to explain how local operational rules, role constraints, exceptions, and handoffs generate global workflow behavior.
